# UGL型数控不落轮镟床轮对测量数据转换工具
## 工具介绍
这是一款**纯浏览器运行、无需安装软件、无需联网上传**的本地数据转换工具，专为 **UGL型数控不落轮镟床** 输出的CSV轮对测量数据设计，可**批量导入CSV文件 → 一键自动生成标准Excel格式记录表**，完全匹配现场作业表格规范。

---

## 核心功能
✅ **批量转换**：一次选择多个CSV文件，自动批量处理  
✅ **格式精准**：导出Excel **100% 匹配官方标准模板**  
✅ **样式完整**：边框、字体、对齐、合并单元格全部自动生成  
✅ **本地运行**：数据不上传服务器，安全保密，不泄露测量数据  
✅ **编码兼容**：支持 GBK / UTF-8 两种CSV编码，不乱码  
✅ **自动定位**：车辆ID、轮对ID自动识别，自动填入对应单元格  
✅ **日志清晰**：处理成功/失败状态实时显示，方便排查问题  

---

## 导出Excel标准格式（完全固定）
- 标题：轮对测量数据记录表
- 包含：1车、2车、3车、4车 完整轮位
- 字段：踏面直径、轮缘厚度、轮缘高度、轮对内侧距、径向圆跳动、轴向圆跳动、QR值
- 格式：黑体字体、居中对齐、全边框、单元格合并、标准列宽

## 使用方法（超简单 3 步）
### 1. 启动工具
将 `轮对数据转换工具.html` 文件，**直接用 Chrome / Edge / 火狐浏览器** 打开即可使用。

### 2. 选择文件
- 点击 **【选择CSV文件】**
- 可**一次多选**镟床导出的 `.csv` 数据文件
- 支持反复添加，文件列表会自动显示

### 3. 开始转换
- 点击 **【开始转换】**
- 等待进度条跑完
- 浏览器会**自动下载生成好的Excel文件**
- 日志区显示处理结果（成功数 / 失败数）
